This is a sample project showing the main features of the Blip Template Engine.


Don't forget that every time the template is changed it will need to be transpiled.

One approach to make this easier is to run blip with --watch on the template directory.

Then use air to run the application, this will restart the application whenever a template is changed.

blip -dir="web/template" --rebuild
In this directory run: ./buildTemplates.sh
Then in another terminal in this directory run air
